## 🔬 核心技术亮点

### 1. 深度学习算法

#### **BERT-BiLSTM-CRF命名实体识别**
```
数学模型: P(y|x) = ∏(i=1 to n) P(yi|x, y1, ..., yi-1)
损失函数: L = -log P(y*|x)
技术特点: 结合预训练模型、序列建模、全局优化
```

#### **注意力机制关系抽取**
```
注意力计算: Attention(Q,K,V) = softmax(QK^T/√dk)V
关系分类: r = argmax(softmax(W_r·[h_e1; h_e2; h_context] + b_r))
技术特点: 多头注意力、实体感知、上下文建模
```

### 2. 知识图谱技术

#### **图谱表示与构建**
```
三元组表示: KG = {(h, r, t) | h, t ∈ E, r ∈ R}
实体对齐: sim(e1, e2) = α·sim_name + β·sim_attr + γ·sim_struct
技术特点: 结构化知识、语义关系、质量控制
```

#### **图数据库存储**
```
Neo4j存储: 节点-关系-属性模型
Cypher查询: MATCH (e1)-[r]->(e2) WHERE ... RETURN ...
技术特点: 高效图查询、ACID事务、水平扩展
```

### 3. 向量检索技术

#### **文档向量化**
```
BGE嵌入: v_doc = BGE(tokenize(document))
分块策略: chunks = split(document, chunk_size=512, overlap=50)
技术特点: 语义理解、高维表示、相似度计算
```

#### **FAISS检索优化**
```
相似度计算: cosine_sim(q, d) = (q·d)/(||q||·||d||)
索引优化: IVF、HNSW、PQ量化
技术特点: 高速检索、内存优化、精度平衡
```

### 4. 混合检索架构

#### **多源融合**
```
混合评分: score_final = λ·score_vector + (1-λ)·score_keyword
结果融合: 图谱检索 + 向量检索 + 关键词检索
技术特点: 优势互补、智能路由、动态权重
```
