# 🏗️ DLR + LangChain-Chatchat 系统架构文档

## 📋 系统概述

DLR (Document Learning & Reasoning) 是一个集成了知识图谱构建和智能问答的完整AI知识管理系统。通过与LangChain-Chatchat的深度集成，提供了从文档处理到智能问答的端到端解决方案。

## 🎯 核心功能模块

### 1. 👥 用户层
- **管理员用户**: 系统管理、用户管理、配置管理
- **普通用户**: 文档上传、图谱查看、智能问答
- **API用户**: 第三方系统集成、自动化调用

### 2. 🖥️ 前端界面层
- **DLR主界面** (http://localhost:8000)
  - 知识图谱管理和可视化
  - 文档上传和处理
  - 用户管理和系统配置
- **Chatchat界面** (http://localhost:8501)
  - 智能问答交互
  - 知识库管理
  - 对话历史查看
- **管理后台** (http://localhost:8000/admin)
  - Django管理界面
  - 数据库管理
  - 系统监控
- **API文档** (http://localhost:7861/docs)
  - 接口文档和测试
  - API调用示例

## 🔄 数据流程

### 1. 知识图谱构建流程
```
文档上传 → 文档解析 → 实体识别 → 关系抽取 → 图谱构建 → 存储到Neo4j → 可视化展示
```

### 2. 智能问答流程
```
用户问题 → 意图识别 → 知识检索 → 大模型推理 → 答案生成 → 返回结果
```

### 3. 超级问答流程
```
用户问题 → 混合检索(图谱+知识库) → 结果融合 → 综合推理 → 精准答案
```
